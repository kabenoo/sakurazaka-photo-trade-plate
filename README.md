# 生写真交換プレート作成アプリ

## 内容
ブラウザだけで動く、横長の生写真交換プレート作成アプリです。

- 「求」「譲」それぞれでメンバー選択と手入力に対応
- 同じ苗字がある場合は、名前の1文字を括弧で表示
- JPEG保存に対応
- スマホ横画面で見やすい 1536×1024 サイズ

## 使い方
1. `index.html` をブラウザで開きます。
2. 「求」「譲」を入力します。
3. `プレート作成` を押します。
4. `JPEG保存` または `スマホで共有・保存` を押します。

スマホでは、保存後に画像を長押し、または共有メニューから写真に保存してください。

## GitHub Pagesで公開する場合
このフォルダ内のファイルをそのままリポジトリにアップロードしてください。

必要なファイルは以下です。

- `index.html`
- `style.css`
- `app.js`
- `manifest.webmanifest`
- `service-worker.js`
- `icon.svg`

## メンバー一覧の更新
メンバー一覧は `app.js` の `MEMBERS` を編集してください。
